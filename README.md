# React Name Form

Um exemplo simples de aplicação **React** que captura o nome do usuário em um formulário e o exibe na tela.

## Como funciona

- O usuário digita seu nome em um `<input>`.
- Ao submeter o formulário, o nome é armazenado em `submittedName`.
- O componente renderiza a mensagem `Olá, <nome>!`.

Tudo isso roda direto no navegador usando React e Babel carregados a partir de CDNs, sem necessidade de build.

## Rodando localmente

```bash
# Clonar o repositório
git clone https://github.com/gsbenevides2/react-name-form.git
cd react-name-form

# Abra o arquivo no navegador (não precisa instalar dependências)
open index.html   # ou abra manualmente no seu navegador
```
